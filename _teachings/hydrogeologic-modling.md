## Course Overview

This introductory course on hydrogeologic modeling covers fundamental concepts and numerical mdethods for groundwater flow and solute transport. By the end of this course, students will be able to:

- Understand groundwater flow mechanisms
- Implement basic finite volume methods for groundwater flow modeling 

## Prerequisites

- Basic knowledge of linear algebra and calculus
- Programming experience in MATLAB (on which course material is developed) or Python
